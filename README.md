## Project Overview
This project analyzes survey data from data professionals using **Power BI**.  
The goal is to understand trends in roles, salaries, skills, and overall job satisfaction in the data field.

I prepared and cleaned the raw data, then built an **interactive Power BI dashboard** that allows users to explore and compare insights across different dimensions.

---

## Dashboard Features
The interactive dashboard allows you to analyze:

-  Average salary by job title  
-  Favorite programming languages  
-  Difficulty of breaking into data roles  
-  Survey takers by country  
-  Happiness with salary  
-  Happiness with work-life balance  
-  Number of survey takers  
-  Average age of survey takers  

Users can filter, compare, and explore the data dynamically.
